Lista de funciones Jquery:
  • Closest(X):Busca y devuelve el primer nodo padre que recibe(X)
  • attr():devuelve el valor que tenga el atributo introducido
  • append(X):mueve el elemento dado X y lo mueve al nodo que reliza la funcion
  • css():devuelve o cambia el valor del atributo css del nodo que realiza la funcion
  • find():devuelve todos los nodos hijos que se introduzca como paramentro del elemento que lo realice
  • remove():elimina el nodo que realice la funcion
  • .each():es el equivalente al for-each de JQUERY,util para que los elementos realicen la misma funcion.
  • .click():establece la funcion dada para que se dispare cuando se clicke el elemento dado
  • .addClass:otorga la clase dada al elemento en cuestion
  • .toggleClass:cambia la clase actual del elemento por la introducida
  • .removeClass():elimina la clase introducida
  • .on():otorga al elemento que lo realice un evento(como keyup) y la funcion que disparará la misma
  • val():recoge o cambia el valor de un atributo al nodo
  • text():recoge o cambia el texto al nodo
  • includes():devuelve un booleano segun SI esta el parametro dado en el elemento que lo realice
  • parseInt():cambia el parametro introducido a uno formato Int
